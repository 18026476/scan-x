# SCAN-X v1.0.0 RC1 Release Notes

SCAN-X is a Windows-based network discovery and security auditing platform designed to help users identify devices, evaluate network exposure, and perform basic security assessments.

## Highlights

- Device discovery and inventory
- Router security auditing
- IoT security analysis
- Security alert engine
- Risk scoring
- Bundled Nmap integration
- Report generation and export

## Validation Completed

- Full Flutter test suite passed
- Windows release build completed successfully
- Clean-machine smoke test passed
- Nmap integration verified
- Settings validation completed
- Alert engine validation completed

## Known Notes

- This is a Release Candidate build.
- Additional real-world testing is encouraged.
- Windows Defender or firewall prompts may appear due to network scanning functionality.

## Usage Warning

Only use SCAN-X on networks and devices that you own or are explicitly authorized to assess.
