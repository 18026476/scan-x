# SCAN-X

SCAN-X is a Windows-based network discovery and security auditing tool designed for home users, enthusiasts, and small businesses.

## Features

- Network device discovery
- Device identification and inventory
- Risk scoring and security assessment
- Router security auditing
- IoT device security checks
- Security alert engine
- Report generation and export
- Bundled Nmap integration
- Windows desktop application

## Installation

1. Download the latest release ZIP from the Releases page.
2. Extract the ZIP to a folder of your choice.
3. Run `scanx_app.exe`.

No Flutter installation is required.

## Requirements

- Windows 10 or Windows 11
- Administrator privileges may be required for some network scanning functions
- Local network access

## Security Notice

SCAN-X is intended for defensive security purposes only.

Only scan networks, devices, and infrastructure that you own or are explicitly authorized to assess.
